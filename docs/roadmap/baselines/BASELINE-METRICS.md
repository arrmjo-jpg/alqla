RUM/CrUX). **لا توجد أداة Real User Monitoring مُفعَّلة في المشروع حاليّاً
لقياس INP الحقيقيّ** — فجوة قياس حقيقيّة، تُذكَر هنا بصراحة بدل افتراض
قيمة لم تُقَس.
